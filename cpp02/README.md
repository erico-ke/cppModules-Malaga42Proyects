# cpp02

Ejercicios centrados en clases, encapsulación y las reglas de copia/assignación.

Proyectos destacados

- `ex02` (`Account`): tracking de operaciones y estados internos (métodos estáticos, encapsulación de datos y funciones miembro).

Objetivos de aprendizaje

- Implementar constructores por defecto, copia y operadores de asignación cuando proceda.
- Manejar correctamente variables estáticas dentro de clases.

Cómo compilar y ejecutar

```bash
cd cpp02/ex02
make
./ex02
```

Notas

- Revisa `Makefile` para flags de compilación y estándar requerido.

Makefile (objetivos comunes)

- Busca `all` (compila), `clean`, `fclean` y `re` en el `Makefile` de cada ejercicio.

Ejemplos de uso

- `ex00` (fixed):
	```bash
	cd cpp02/ex00
	make
	./fixed
	```
- `ex01` (moreUsefulFixed):

- `ex02` (moreFixed):
